🏆 Blind Auction (Python CLI)


A simple Python program that simulates a blind auction where multiple users place bids, and the highest bidder wins.

🚀 How it works...
Users enter their name and bid amount
Multiple bidders can participate
Input continues until no more bidders join
Program finds and displays the highest bidder

💻 Tech Used...
Python 3

🧠 What I learned...
Dictionaries in Python
Loops and condition handling
Basic problem-solving logic

🔥 Output...
Shows the winner with the highest bid at the end.
